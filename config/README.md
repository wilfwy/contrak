# Configuration Firebase

Placez votre fichier `firebase-service-account.json` dans ce dossier.

Ce fichier est généré depuis la Firebase Console :
1. Allez dans Paramètres du projet > Comptes de service
2. Cliquez sur "Générer une nouvelle clé privée"
3. Téléchargez le fichier JSON
4. Renommez-le en `firebase-service-account.json` et placez-le ici

**⚠️ IMPORTANT : Ce fichier ne doit jamais être committé dans Git !**
